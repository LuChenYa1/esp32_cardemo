## 组件简介

BLE 是一个蓝牙低功耗（Bluetooth Low Energy）通信组件，用于实现 ESP32 的 BLE 功能。该组件当前处于预留状态，头文件已创建但功能尚未实现。

### 主要特性

- 预留 BLE 功能接口
- 待实现：BLE 广播、扫描、连接、数据传输

### 适用场景

- BLE 设备通信
- 低功耗无线数据传输
- 移动设备连接

---

## 硬件连接

### 引脚分配

BLE 使用 ESP32 内置蓝牙硬件，无需外部引脚连接。

### 注意事项

- ⚠️ BLE 与 WiFi 共用 2.4GHz 射频，可能存在干扰
- ⚠️ 使用 BLE 时需要配置 ESP-IDF 的 Bluetooth 组件
