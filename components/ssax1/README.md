## 组件简介

SSA接口模块提供对SSA（Smart Sensor Array）接口的GPIO读取功能。该模块通过轮询方式读取多个GPIO引脚的状态，适用于数字传感器阵列的数据采集。

### 主要特性

- 支持多个GPIO引脚的状态读取
- 轮询方式采集数据
- 简单的初始化和读取接口
- 适用于数字传感器阵列

### 适用场景

适用于需要读取多个数字传感器状态的场景，如传感器阵列、开关状态检测、数字信号采集等。

## 硬件连接

### 引脚分配

| 功能 | GPIO引脚 | 接口标识 | 说明 |
|------|---------|---------|------|
| SSA1 | 可配置 | SSA1 | 传感器阵列信号1 |
| SSA2 | 可配置 | SSA2 | 传感器阵列信号2 |
| SSA3 | 可配置 | SSA3 | 传感器阵列信号3 |
| ... | ... | ... | 根据实际需求配置 |

### 接线说明

1. 将传感器阵列的信号线连接到ESP32的GPIO引脚
2. 确保传感器的电源和地线正确连接
3. 根据传感器类型配置上拉/下拉电阻

### 注意事项

- ⚠️ **引脚配置**：需要在代码中配置具体的GPIO引脚号
- ⚠️ **电平兼容**：确保传感器输出电平与ESP32兼容（3.3V）
- ⚠️ **上拉/下拉**：根据传感器类型配置合适的上拉或下拉电阻

---

## 功能说明

### 核心功能

#### 功能1：GPIO初始化

配置SSA接口使用的GPIO引脚为输入模式，设置上拉/下拉电阻。

#### 功能2：状态轮询

周期性读取所有GPIO引脚的状态，并打印到串口。

#### 功能3：数据读取

提供简单的读取接口，获取当前GPIO状态。

## 使用示例

### 基本使用示例

```c
#include "ssax1.h"
#include "freertos/FreeRTOS.h"
#include "freertos/task.h"

void ssax1_example_task(void *pvParameters)
{
    // 1. 初始化SSA接口
    ssax1_gpio_init();
    
    // 2. 循环读取GPIO状态
    while (1) {
        ssax1_poll_gpio_status();
        vTaskDelay(pdMS_TO_TICKS(100));  // 100ms轮询间隔
    }
}

void app_main(void)
{
    xTaskCreate(ssax1_example_task, "ssax1_task", 2048, NULL, 5, NULL);
}
```

## 注意事项

### 硬件限制

- ⚠️ **引脚数量**：受ESP32可用GPIO数量限制
- ⚠️ **电平兼容**：传感器输出必须为3.3V电平或使用电平转换

### 软件限制

- ⚠️ **轮询方式**：采用轮询方式，不使用中断，占用一定CPU时间
- ⚠️ **引脚配置**：需要在源代码中配置具体的GPIO引脚号

### 性能考虑

- 轮询间隔影响响应速度和CPU占用
- 建议根据实际需求调整轮询频率
