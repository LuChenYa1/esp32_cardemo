## 组件简介

WiFi 是一个基于 ESP-IDF WiFi 驱动的无线网络连接组件，提供 Station 模式的 WiFi 连接功能。该组件封装了 WiFi 初始化、连接、事件处理等功能，简化 WiFi 使用流程。

### 主要特性

- WiFi Station 模式
- 自动连接和重连
- 事件驱动机制
- 获取 IP 地址
- 时间同步（可选）

### 适用场景

- 需要连接到 WiFi 网络的应用
- 物联网设备联网
- 远程控制和监控
- 数据上传和下载

---

## 硬件连接

### 引脚分配

WiFi 使用 ESP32 内置 WiFi 硬件，无需外部引脚连接。

### 注意事项

- ⚠️ WiFi 与 BLE 共用 2.4GHz 射频，可能存在干扰
- ⚠️ WiFi 功耗较高，电池供电设备需要考虑功耗管理

---

## 功能说明

### 核心功能

#### 功能1：WiFi 连接

连接到指定的 WiFi 网络（SSID 和密码在头文件中配置）。

#### 功能2：事件处理

自动处理 WiFi 连接、断开、获取 IP 等事件。

#### 功能3：时间同步

可选的时间打印任务，用于调试和时间同步。

### 配置参数

| 参数名称 | 默认值 | 取值范围 | 说明 |
|---------|--------|---------|------|
| WIFI_SSID | "slips" | - | WiFi名称 |
| WIFI_PASS | "88888888" | - | WiFi密码 |

## API接口

### 初始化函数

```c
/**
 * @brief 初始化WiFi Station模式
 */
void wifi_init_sta(void);
```

**参数说明**：
- 无

**返回值**：
- 无

**使用说明**：
调用此函数初始化 WiFi 并连接到配置的网络。连接成功后会自动获取 IP 地址。

## 使用示例

### 基本使用示例

```c
#include "wifi.h"
#include "esp_log.h"

static const char *TAG = "WIFI_EXAMPLE";

void app_main(void)
{
    // 初始化WiFi并连接
    wifi_init_sta();
    
    ESP_LOGI(TAG, "WiFi初始化完成，等待连接...");
    
    // 后续可以使用TCP、UDP等网络功能
}
```
