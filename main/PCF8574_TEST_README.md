# PCF8574 组件测试程序

## 概述

本测试程序用于验证独立的 `pcf8574` 组件功能，确保其可以正常替代 `managed_components` 中的 pcf8574 驱动。

## 测试内容

### 1. I2C 总线扫描
- 扫描地址范围 0x00-0x7F
- 检测并显示所有响应的 I2C 设备
- 验证 PCF8574 是否在线（默认地址 0x20）

### 2. 端口读写测试
- 测试 8 位并行读写功能
- 写入不同的测试值：0x00, 0xFF, 0xAA, 0x55, 0x0F, 0xF0
- 读取并验证写入的值
- 以二进制格式显示结果

### 3. 单引脚读写测试
- 依次测试 P0-P7 每个引脚
- 设置高电平并读取验证
- 设置低电平并读取验证
- 验证单引脚操作的准确性

### 4. LED 流水灯效果
- P0-P7 依次点亮
- 正向流水 + 反向流水
- 循环 5 轮
- 验证引脚切换速度

### 5. PWM 呼吸灯效果
- 在 P2 引脚上实现软件 PWM
- 亮度从 0% 渐变到 100%，再从 100% 渐变到 0%
- 循环 10 个周期
- 验证 I2C 通信速度是否满足 PWM 需求

## 硬件连接

### PCF8574 接线

| PCF8574 引脚 | ESP32 引脚 | 说明 |
|-------------|-----------|------|
| VCC         | 3.3V      | 电源 |
| GND         | GND       | 地   |
| SDA         | GPIO21    | I2C 数据线 |
| SCL         | GPIO20    | I2C 时钟线 |
| A0-A2       | GND       | 地址设置（0x20）|

### LED 连接（可选）

| PCF8574 引脚 | 连接方式 | 说明 |
|-------------|---------|------|
| P0-P7       | LED + 限流电阻 -> GND | 低电平点亮 |

或者

| PCF8574 引脚 | 连接方式 | 说明 |
|-------------|---------|------|
| P0-P7       | VCC -> 限流电阻 + LED | 高电平点亮 |

## 编译和运行

### 1. 配置项目

确保 `main/CMakeLists.txt` 中包含测试文件：

```cmake
idf_component_register(
    SRCS "main_pcf8574_test.c"
    INCLUDE_DIRS "."
    REQUIRES pcf8574
)
```

### 2. 编译

```bash
idf.py build
```

### 3. 烧录

```bash
idf.py flash
```

### 4. 查看日志

```bash
idf.py monitor
```

## 故障排查

### 问题 1：I2C 扫描未发现设备

**可能原因：**
- PCF8574 未正确供电
- I2C 总线缺少上拉电阻
- 接线错误（SDA/SCL 接反）
- 地址设置错误

**解决方法：**
1. 检查 VCC 和 GND 连接
2. 添加 4.7kΩ 上拉电阻到 SDA 和 SCL
3. 确认 SDA=GPIO21, SCL=GPIO20
4. 检查 A0-A2 引脚接地（地址 0x20）

### 问题 2：读写测试失败

**可能原因：**
- I2C 通信不稳定
- 电源电压不足
- 总线干扰

**解决方法：**
1. 降低 I2C 频率（修改为 100kHz）
2. 使用稳定的 3.3V 电源
3. 缩短连接线长度
4. 添加去耦电容（0.1uF）

### 问题 3：PWM 呼吸灯效果不明显

**可能原因：**
- I2C 通信速度限制
- LED 连接方式不正确

**解决方法：**
1. 这是正常现象，PCF8574 的 I2C 通信速度限制了 PWM 频率
2. 确认 LED 连接方式（低电平点亮或高电平点亮）
3. 调整占空比变化步长

## 与 managed_components 版本的对比

| 特性 | managed_components | 独立组件 |
|------|-------------------|---------|
| 依赖 | i2cdev, esp_idf_lib_helpers | 仅 driver |
| I2C 驱动 | 兼容旧版 | ESP-IDF 5.3+ 新驱动 |
| API 接口 | i2c_dev_t | i2c_dev_t（兼容）|
| 初始化 | pcf8574_init_desc | pcf8574_init_desc（相同）|
| 读写 | pcf8574_port_read/write | pcf8574_port_read/write（相同）|
